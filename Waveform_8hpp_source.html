Waveform
Waveform is a C++ header-only library which represents both the time and the frequency domains of a waveform/signal as a single object, transforming using FFTW automatically when needed.

626  // Arithmetic Operator Overloads
628  // Because Waveform requires two containers of floating-based types,
629  // we assume that the arithmetic operators work element-wise on these
630  // containers.
631  //
632  // It would be ideal to be able to define the actual operators as a
633  // iterator-based implementation, such as:
634  //
635  // operator+= (rhs)
636  // {
637  // std::transform( cbeginTime()
638  // , cendTime()
639  // , rhs.cbeginTime()
640  // , beginTime()
641  // , std::plus<TimeT, FreqT>()
642  // );
643  // return *this;
644  // }
645  //
646  // However, doing this would prevent specialization of the operators
647  // at a container level. For example, DiscreteSeries needs to be able
648  // to determine whether or not the time bins are the same when summing
649  // two timeSeries objects.
650  //
651  // My solution to this is to require that any container (such as
652  // std::vector)also have the appropriate operators defined. Boost
653  // has definitions in boost/accumulators/numeric/functional/vector.hpp
654  // in the namespace boost::numeric::operators.
